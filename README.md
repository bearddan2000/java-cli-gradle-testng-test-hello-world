# java-cli-gradle-testng-test-hello-world

## Description
A POC for gradle app using TestNG.

## Tech stack
- java
- gradle
  - TestNG

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
